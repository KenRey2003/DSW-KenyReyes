Sintaxis y Selectores Css
Propósito del sitio:
  • Seleccionar todos los elementos de las páginas donde la etiqueta HTML coincide con el valor del selector seleccionando la etiqueta HTML (sin las características) o cuando se quiera dar el mismo estilo a dos elementos se puede encadenar, agrupar las reglas de css con un selector múltiple separado con una coma agrupar las propiedades con una única regla css y definir propiedades comunes(color y tipo de letra)
    Introducción
  • Los selectores Css se emplean para elegir los elementos de HTML y darle estilo a la web hay muchos selectores CSS que sirven para seleccionar los elementos adecuados ellos se dividen en varios categorías las cuales nos indican que elementos usar en ciertas características de la página para organizarla de una manera correcta
